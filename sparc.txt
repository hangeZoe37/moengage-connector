POST

/backend/api/v2/rcs/sendmessage

Send RCS Message

Send RCS messages using API-based messages. This endpoint supports Plain Text, Rich Text, Rich Card, and Rich Card Carousel messages.

Messages can be sent using templates with support for:

Variables

Suggestions

Media (image / carousel)

Delivery report callbacks (DLR)

Parameters

Try it out

No parameters

Request body

application/json

Examples: Plain TextRich Text with Suggestions & VariablesRich CardRich Card Carousel

Example Value

Schema

{

  "messages": [

    {

      "template_name": "testtempl1103",

      "ttl": "300s",

      "addresses": [

        {

          "assistant_id": "68a2f70eec220d7409d8139c",

          "mobile_number": "919390977728"

        }

      ]

    }

  ],

  "dlr_url": [

    {

      "url": "https://example.com/"

    }

  ]

}
